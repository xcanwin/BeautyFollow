# BeautyFollow
###Purpose
```
Followers of books and TV series.
```

###usage:
```
usage: BeautyFollow.py [-h] [-r] [-U] [-u UPDATE] [-t TIMEOUT]

optional arguments:
  -h, --help  show this help message and exit
  -r          read list;
  -U          update all;
  -u UPDATE   update anyone by id;
  -t TIMEOUT  request timeout (default: 20);
```
